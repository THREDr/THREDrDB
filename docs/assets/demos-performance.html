These demos showcase the performance and efficiency gains possible with THREDrDB. See how a single SQL query can replace dozens of traditional Excel lookup formulas while being easier to read and maintain.

🔍 VLOOKUP vs QUERY — Lookup Showdown
Demonstrates how a single THREDrDB.QUERY with a SQL JOIN replaces an entire grid of VLOOKUP formulas. A 100-row employee table is looked up by 20 employee IDs — traditional Excel needs 80 VLOOKUPs (4 columns × 20 rows), while THREDrDB does it with one formula.
  • Creates a 100-row Employees lookup table (ID, Name, Department, Salary, Location)
  • Creates a 20-row LookupIDs table with selected employee IDs
  • Places 80 VLOOKUP formulas (4 per row × 20 rows) to retrieve each field
  • Places a single THREDrDB.QUERY that joins both tables and returns all results
  • Compare side-by-side: 80 formulas vs 1 formula, same result
