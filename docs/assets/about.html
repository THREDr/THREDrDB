What is THREDrDB?

THREDrDB (Threaded Database) is a powerful Excel add-in that brings database functionality directly into your spreadsheets. It allows you to create in-memory databases, query data using SQL, and seamlessly integrate external data sources with your Excel workbooks.

Key Features

  • Create in-memory databases from Excel ranges, CSV files, and API calls
  • Query data using standard SQL syntax
  • Real-time data updates and calculations
  • Import and manage external data sources
  • Export data to CSV and XLSX formats
  • AI-powered Copilot integration for query assistance
  • Automatic schema generation and documentation
  • Cross-platform support (Windows, Mac, Excel Online)

Technology

THREDrDB is built using modern web technologies and integrates seamlessly with Microsoft Office:

  • Office.js: Microsoft Office JavaScript API
  • TypeScript: Type-safe development
  • DuckDB: High-performance SQL database engine
  • Webpack: Modern build tooling
