EXPORTQUERY

Description: Exports the result of a SQLite query to a downloadable file (CSV or JSON).

Syntax: =THREDRDB.EXPORTQUERY(query, [fileType], [fileName], [incHeaders], [databaseName], [debugging])

  • query (string): SQLite query to execute and export
  • fileType (string, optional): File type to export ("csv" or "json")
  • fileName (string, optional): Name of the exported file
  • incHeaders (boolean, optional): Include headers in the export (true/false)
  • databaseName (string, optional): Optional workbook path (uses current workbook if empty)
  • debugging (boolean, optional): Show error messages for debugging (true/false)

Returns: Download link or status (string)

REFRESHTABLE

Description: Refreshes a table in the SQLite database by re-importing its source data.

Syntax: =THREDRDB.REFRESHTABLE(tableName, [databaseName], [debugging])

  • tableName (string): Name of the table to refresh
  • databaseName (string, optional): Optional workbook path (uses current workbook if empty)
  • debugging (boolean, optional): Show error messages for debugging (true/false)

Returns: Status message (string)

QUERY

Description: Runs a SQLite query on the THREDr database and returns the result as a matrix.

Syntax: =THREDRDB.QUERY(query, [incHeaders], [databaseName], [debugging], [queryDefinition], [returnDefinition], [returnRows], [prerequisiteRange])

  • query (string): SQLite query to execute
  • incHeaders (boolean, optional): Include headers in the result (true/false)
  • databaseName (string, optional): Optional workbook path (uses current workbook if empty)
  • debugging (boolean, optional): Show error messages for debugging (true/false)
  • queryDefinition (string, optional): Optional name to save this query as a view
  • returnDefinition (boolean, optional): Return the query definition name instead of data (true/false)
  • returnRows (boolean, optional): Return row count instead of data (true/false)
  • prerequisiteRange (any, optional): Optional range reference to ensure calculation dependencies

Returns: Query result (matrix)

VOLATILEQUERY

Description: Runs a volatile SQLite query on the THREDr database (recalculates frequently).

Syntax: =THREDRDB.VOLATILEQUERY(query, [incHeaders], [databaseName], [debugging], [queryDefinition], [returnDefinition], [returnRows], [prerequisiteRange])

  • query (string): SQLite query to execute
  • incHeaders (boolean, optional): Include headers in the result (true/false)
  • databaseName (string, optional): Optional workbook path (uses current workbook if empty)
  • debugging (boolean, optional): Show error messages for debugging (true/false)
  • queryDefinition (string, optional): Optional name to save this query as a view
  • returnDefinition (boolean, optional): Return the query definition name instead of data (true/false)
  • returnRows (boolean, optional): Return row count instead of data (true/false)
  • prerequisiteRange (any, optional): Optional range reference to ensure calculation dependencies

Returns: Query result (matrix)

CLOCK

Description: Displays the current time on a regular interval.

Syntax: =THREDRDB.CLOCK([intervalSeconds])

  • intervalSeconds (number, optional): Update interval in seconds (default: 1 second)

Returns: Current time (string, updates automatically)

PROCESSJSON

Description: Processes a JSON string and returns a matrix or value based on the provided path.

Syntax: =THREDRDB.PROCESSJSON(jsonString, [jsonPath], [asMatrix])

  • jsonString (string): JSON string to process
  • jsonPath (string, optional): JSONPath expression to extract data
  • asMatrix (boolean, optional): Return result as matrix (true/false)

Returns: Extracted value or matrix

APICALL

Description: Makes an HTTP request and optionally loads data into the database.

Syntax: =THREDRDB.APICALL(url, [tableName], [databaseName], [prerequisiteRange], [apiName], [cacheTimeoutSeconds])

  • url (string): API URL to call
  • tableName (string, optional): Table processing (see docs)
  • databaseName (string, optional): Optional workbook path (uses current workbook if empty)
  • prerequisiteRange (any, optional): Optional range reference to ensure calculation dependencies
  • apiName (string, optional): Optional API name prefix for tables
  • cacheTimeoutSeconds (number, optional): Cache timeout in seconds (default: 600 = 10 minutes)

Returns: API result (matrix)

APICALLADV

Description: Advanced API call with custom HTTP method, headers, and body.

Syntax: =THREDRDB.APICALLADV(url, [method], [headers], [body], [tableName], [databaseName], [prerequisiteRange], [apiName], [cacheTimeoutSeconds])

  • url (string): API URL to call
  • method (string, optional): HTTP method (GET, POST, etc.)
  • headers (string, optional): HTTP headers as JSON string
  • body (string, optional): HTTP request body
  • tableName (string, optional): Table processing (see docs)
  • databaseName (string, optional): Optional workbook path (uses current workbook if empty)
  • prerequisiteRange (any, optional): Optional range reference to ensure calculation dependencies
  • apiName (string, optional): Optional API name prefix for tables
  • cacheTimeoutSeconds (number, optional): Cache timeout in seconds (default: 600 = 10 minutes)

Returns: API result (matrix)

IMPORT

Description: Imports data from a supported file (CSV, XLSX, JSON) into the SQLite database.

Syntax: =THREDRDB.IMPORT(filePath, [tableName], [databaseName], [options], [debugging])

  • filePath (string): Path or URL to the file to import
  • tableName (string, optional): Name of the table to import into
  • databaseName (string, optional): Optional workbook path (uses current workbook if empty)
  • options (string, optional): Import options as JSON string
  • debugging (boolean, optional): Show error messages for debugging (true/false)

Returns: Status message (string)

SUMFIELD

Description: Sums a field from a SQLite database table based on criteria.

Syntax: =THREDRDB.SUMFIELD(tableName, fieldName, [whereClause], [databaseName], [debugging])

  • tableName (string): Table name to sum from
  • fieldName (string): Field name to sum
  • whereClause (string, optional): Optional WHERE clause for filtering
  • databaseName (string, optional): Optional workbook path (uses current workbook if empty)
  • debugging (boolean, optional): Show error messages for debugging (true/false)

Returns: Sum (number)

LOOKUP

Description: Looks up a value from a SQLite database table.

Syntax: =THREDRDB.LOOKUP(tableName, returnField, whereClause, [databaseName], [debugging])

  • tableName (string): Table name to lookup from
  • returnField (string): Field name to return
  • whereClause (string): WHERE clause for matching
  • databaseName (string, optional): Optional workbook path (uses current workbook if empty)
  • debugging (boolean, optional): Show error messages for debugging (true/false)

Returns: Lookup result (any)

GETTABLELIST

Description: Gets a list of tables in the database.

Syntax: =THREDRDB.GETTABLELIST([databaseName])

  • databaseName (string, optional): Optional workbook path (uses current workbook if empty)

Returns: Table list (matrix of strings)

GETCOLUMNLIST

Description: Gets a list of columns in a specific table.

Syntax: =THREDRDB.GETCOLUMNLIST(tableName, [databaseName])

  • tableName (string): Name of the table to get columns from
  • databaseName (string, optional): Optional workbook path (uses current workbook if empty)

Returns: Column list (matrix of strings)

BUILDVERSION

Description: Returns the current build version timestamp.

Syntax: =THREDRDB.BUILDVERSION()

Returns: Build version (string)
